# 读书打卡第二十四天

## 毛姆《在中国屏风上》

### 五十六 副领事

宁做太平犬，莫作离乱人。乱世，不光让无数人莫名其妙丢掉性命，同时让生者麻木，眼看死亡而无动于衷，这是比死亡还要可怕的事情。

副领事，一个小年轻，会因为站在领导面前而紧张，会因为看见囚犯苍白的脸而窘迫、不好意思。同时也会在枪毙囚犯之后漫不经心地说起这个事情“跟往常一样”。

### 五十八 祭神

这个老妇人是在祭神吗？祭神不是应该在寺庙，在年底几个大节日吗？我看更像是在祭祖。一来是在路边，二来烧了纸钱。
