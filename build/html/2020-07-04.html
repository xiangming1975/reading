# 读书打卡第二十五天

## 毛姆《客厅里的绅士》

又是一本游记，这次是东南亚。和《在中国屏风上》相比，这本书各章节之间的联系相当紧密。《屏风》一书可以说是一串珍珠，各章节调换顺序并不影响阅读，但这本书不行，各章节是按照行程来写的，所以也得老老实实按顺序读下去。

就我的感观而言，东南亚于我，相当疏离，也就没了读《屏风》时的代入感，只是安安静静地跟着毛姆一路走一路看。事实上，毛姆也确实在专心旅游，领略人文和自然风情，并没有像在中国那样拜会社会各界人物。

毛姆的游记，不光好在他的文笔，也好在他的心态。

我见过太多旅人抱着高人一等的心态去旅行。我比读者有钱有闲，我赏脸带着你们旅行，这些地方你们没来过吧；我比土著摩登，我来你们这儿旅游只是来猎奇；我比其他游人优越，旅游途中我文明我高尚我礼貌……

这些毛病毛姆全都没有。毛姆全程以平和的心态去看待旅途中遇到的形形色色人物。他不带偏见，尤其不带西人惯有的自大，温柔地感受途中遇到的男男女女的悲喜，无论是土著，还是其他旅人。

毛姆最关怀的永远都是人，无论他是在旅途中，还是在当特工时，都体现着对人性的无尽关怀，哪怕他当时在嘲讽在针刺。这是我喜欢毛姆的唯一原因。
